# Route Assignment 2 - Gym Landing Page

A gym landing page developed using HTML and CSS as part of the Route Front-End Development course.

## Features

* Hero section with promotional content and call-to-action button.
* Why Choose Us section highlighting gym advantages.
* Testimonials section displaying customer feedback.
* Membership registration form for potential members.

## Technologies Used

* HTML5
* CSS3

## Learning Objectives

This project focuses on practicing page structure, layout design, positioning, floats, inline-block elements, spacing, typography, and styling without using advanced layout techniques such as Flexbox or Grid.
